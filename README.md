# Python Calculator

A simple calculator built with Python.

## How to Run
1. Make sure Python is installed
2. Run this command:

python_calculator.py
## Operations Supported
- Addition ➕
- Subtraction ➖
- Multiplication ✖️
- Division ➗

## Then upload it
git add README.md
git commit -m "added README"
git push origin master
